# Fake Tour Guide: Pompeii Edition

Javier se hace pasar por guia turistico de Pompeii con datos completamente inventados.

## Contents

- `fake-facts-es.md` — Datos falsos en espanol (script principal)
- `fake-facts-en.md` — Fake facts in English (reference)
- `fake-facts-it.md` — Fatti falsi in italiano (per sembrare autentico)
- `image-prompts.md` — Prompts for generating terrible low-res "visual aids"

## How it works

Javier gets a laminated badge, a folder of blurry printed images, and a script of
"facts" that start plausible and escalate into complete nonsense. He presents these
to the group (and nearby tourists) with full confidence.
